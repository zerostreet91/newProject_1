CSS Position property reference

What is CSS?

CSS is the language we use to style an HTML document.
CSS describes how HTML elements should be displayed.

How to position your content box

Property Value Description
Position Static Default position property. Isn't affected by directional properties.
Relative Postion itself from the original position to move accordingly to the directional properties.
fixed Removes the content box from the normal flow and viewport and then fix it to the position of the given property(Top, bottom, left, right) it stays visable even if the page is scrolled.
Absolute Removes the content box from the normal flow but not the viewpoint. The content box can be positioned relative to the closest ancestor(parent) content box. Doesn't stay when scrolled.
It will overlap other content. Z-index is recommended with absolute position.
Sticky This position property is bit like fixed, however where as fixed position is not effected by scrolling, the sticky position is toggled once it reaches its' scrolling position(when the borders meet) and then sticks in place by the given property(top,bottom, left,right).

How to change and fix your content box size

Property Value Description
Height & Width auto This is the default setting for the height and width size of the content box. The box height and width is automatically calculated and assigned by the browser.
"Length" There is no such css value declaration as "length". This is the value where you input a number value with the unit you want to use. Defining the height or width in px(pixel), cm, in etc.
